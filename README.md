# IC_PY
Implementação em Python do Fluxo de Potência Ótimo Reativo solucionado pelo Método Primal-Dual de Pontos Interiores (sistemas IEEE 14, 30 e 57 barras). O repositório contém o código-fonte do método numérico e diferentes Jupyter Notebooks voltados para a otimização dos cálculos e análise de convergência.
